DomOutline - 基础使用示例

点击按钮开始/停止 DOM 元素检查，然后移动鼠标查看元素轮廓

idle

功能特性

  • ✅ 鼠标悬停显示元素轮廓
  • ✅ 点击元素获取详细信息
  • ✅ 键盘导航（Ctrl+↑/↓）
  • ✅ 支持 iframe 和 shadow DOM
  • ✅ 可配置排除选择器

使用说明

  • 点击"开始检查"按钮启动检查器
  • 移动鼠标到不同元素上查看轮廓
  • 点击元素获取详细信息
  • 使用 Ctrl+↑/↓ 导航到父/子元素
  • 点击"停止检查"结束检查
